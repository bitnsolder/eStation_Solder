# eStation_Solder
An open-source electronic soldering station (eStation) for JBC C245 tips, powered by STM32. Features custom firmware and hardware design. Part of Bit &amp; Solder.


**Note:** This repository temporarily reuses a few functions from the AxxSolder project to accelerate initial development. These dependencies fully comply with the GPLv3 license and may be independently optimized or rewritten in future updates to better fit this specific hardware architecture.
